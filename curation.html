Curation standards

How we curate

Built for discovery. Reviewed before display.

CaribbeanSaaS reviews each listing for a genuine Caribbean connection, a clear digital-product purpose, credible public information, and fit for a professional public showcase.

Caribbean connection

The product, team, company, origin, market, or build location must have a meaningful regional connection.

Clear public evidence

We need an official web presence and enough public information to describe the product accurately.

Independent review

Inclusion is not paid placement, an endorsement, a ranking, or a guarantee that a product remains available.

Listed and unlisted records

The directory displays only records whose visibility is listed. The intentionally public catalog data can also retain public-safe unlisted records for ecosystem research without presenting them as active directory listings.

An unlisted record is not a rejection, product-availability conclusion, or endorsement signal. It means only that the record is not currently displayed as an active CaribbeanSaaS listing.
